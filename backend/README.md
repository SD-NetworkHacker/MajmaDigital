
# ⚠️ Backend Obsolète

Ce dossier `backend/` contenait l'ancienne API Node.js/Express.
Puisque le projet a migré vers **Supabase** (Backend-as-a-Service) et **Vercel** (Frontend Hosting), ce code n'est plus utilisé.

Toute la logique métier se trouve désormais dans le dossier `src/services` du frontend, interagissant directement avec Supabase.

Vous pouvez supprimer ce dossier en toute sécurité.
